# สแกนเลขสลาก 6 หลัก (OCR)

เว็บแอปมือถือ เปิดกล้องแล้วสแกนสลากกินแบ่งรัฐบาลสด ๆ → OCR อ่านเลข 6 หลักตัวใหญ่ต่อเนื่อง →
เพิ่มเข้ารายการอัตโนมัติเมื่ออ่านซ้ำได้ตรงกัน → ดาวน์โหลดเป็น CSV ทำงานบนเครื่อง (client-side)
ทั้งหมด ไม่มีระบบล็อกอิน ไม่มีเซิร์ฟเวอร์เก็บข้อมูล รายการที่สแกนจะถูกเก็บไว้ใน localStorage
ของเบราว์เซอร์จนกว่าจะกด "ล้างทั้งหมด" หรือดาวน์โหลด CSV

หมายเหตุ: QR/บาร์โค้ดเล็กบนสลากเป็นรหัสเข้ารหัสเฉพาะของสำนักงานสลากกินแบ่งรัฐบาล
ไม่มี spec สาธารณะให้ถอดเป็นเลข 6 หลักโดยตรง จึงใช้วิธีอ่านตัวเลขที่พิมพ์ด้วย OCR แทน

## เริ่มต้นใช้งาน (local)

```bash
npm install
npm run dev
```

เปิด http://localhost:3000

## วิธีใช้

1. กด "เปิดกล้อง เริ่มสแกน" แล้วอนุญาตให้เว็บใช้กล้อง
2. จัดแถวเลข 6 หลักตัวใหญ่บนสลากให้อยู่ในกรอบเขียว ถือนิ่ง ๆ สักครู่
3. ระบบจะอ่านซ้ำจนมั่นใจแล้วเพิ่มเข้ารายการเองพร้อมสั่นเตือน (มือถือที่รองรับ)
4. ถ้าอ่านผิดหรืออยากเพิ่มเอง พิมพ์เลข 6 หลักในช่อง "หรือพิมพ์เลขเอง" แล้วกด "เพิ่ม"
5. ทำซ้ำทีละใบ จากนั้นกด "ดาวน์โหลด CSV"

> ต้องเปิดผ่าน HTTPS หรือ `localhost` เท่านั้น (ข้อกำหนดของเบราว์เซอร์สำหรับการขอใช้กล้อง)
> เมื่อ deploy ขึ้น Vercel แล้วจะเป็น HTTPS อัตโนมัติ ใช้งานได้ปกติ

## Deploy ขึ้น Vercel

1. สร้าง GitHub repo ใหม่ (ว่างเปล่า) แล้ว push โค้ดนี้ขึ้นไป:
   ```bash
   git init
   git add .
   git commit -m "init lottery scan app"
   git branch -M main
   git remote add origin <URL ของ repo ใหม่>
   git push -u origin main
   ```
2. เข้า https://vercel.com/new → เลือก repo นี้ → Deploy (ไม่ต้องตั้งค่า env var ใดๆ)
