# สแกนเลขสลาก 6 หลัก (OCR)

เว็บแอปมือถือ ถ่ายรูปสลากกินแบ่งรัฐบาล → OCR อ่านเลข 6 หลักตัวใหญ่ → ยืนยัน/แก้ไข →
ดาวน์โหลดเป็น CSV ทำงานบนเครื่อง (client-side) ทั้งหมด ไม่มีระบบล็อกอิน ไม่มีเซิร์ฟเวอร์เก็บข้อมูล
รายการที่สแกนจะถูกเก็บไว้ใน localStorage ของเบราว์เซอร์จนกว่าจะกด "ล้างทั้งหมด" หรือดาวน์โหลด CSV

หมายเหตุ: QR/บาร์โค้ดเล็กบนสลากเป็นรหัสเข้ารหัสเฉพาะของสำนักงานสลากกินแบ่งรัฐบาล
ไม่มี spec สาธารณะให้ถอดเป็นเลข 6 หลักโดยตรง จึงใช้วิธีอ่านตัวเลขที่พิมพ์ด้วย OCR แทน

## เริ่มต้นใช้งาน (local)

```bash
npm install
npm run dev
```

เปิด http://localhost:3000

## วิธีใช้

1. กด "ถ่ายรูป / เลือกรูปสลาก" → ถ่ายภาพเฉพาะแถวเลข 6 หลักตัวใหญ่ให้ชัดและเต็มเฟรม
2. ลากกรอบสี่เหลี่ยมครอบเฉพาะแถวเลข แล้วกด "อ่านเลข (เฉพาะกรอบ)"
3. เลือกเลขที่ระบบอ่านได้ หรือพิมพ์เองถ้าอ่านผิด แล้วกด "เพิ่ม"
4. ทำซ้ำทีละใบ จากนั้นกด "ดาวน์โหลด CSV"

## Deploy ขึ้น Vercel

1. สร้าง GitHub repo ใหม่ (ว่างเปล่า) แล้ว push โค้ดนี้ขึ้นไป:
   ```bash
   git init
   git add .
   git commit -m "init lottery scan app"
   git branch -M main
   git remote add origin <URL ของ repo ใหม่>
   git push -u origin main
   ```
2. เข้า https://vercel.com/new → เลือก repo นี้ → Deploy (ไม่ต้องตั้งค่า env var ใดๆ)
